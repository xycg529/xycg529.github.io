发表于 2022-04-28 更新于 2022-05-18 分类于 考研

一、积分运算

几何应用

例题参考$660$上的第$603$题、第$604$题。

（一）平面面积

平面上任一区域$D$的面积$S_D=\iint\limits_{D}1d\sigma$。

  1. 平面区域$D$由曲线$y=f(x),y=g(x)(f(x)\ge g(x)),x=a,x=b(a<b)$所围成：

    $S_D=\iint\limits_{D}1d\sigma =\int_{a}^{b}dx\int_{g(x)}^{f(x)} dy=\int_{a}^{b}[f(x)-g(x)]dx$

  2. 平面区域$D$由曲线$\rho=\rho (\theta),\theta=\alpha,\theta=\beta(\alpha<\beta)$所围成：

    $S_D=\iint\limits_{D}1d\sigma =\int_{\alpha}^{\beta}d\theta\int_{0}^{\rho(\theta)} d\rho=\frac {1}{2}\int_{\alpha}^{\beta}\rho ^2(\theta)d\theta$

（二）旋转体体积

平面上任一区域$D$绕任一固定轴线$ax+by=c$旋转的体积$V_D=2\pi\iint\limits_{D} r(x,y)d\sigma$，其中$r(x,y)=\frac {|ax+by-c|}{\sqrt{a^2+b^2}}$是$d\sigma$到轴线的距离。

  1. 平面区域$D$由曲线$y=f(x),(f(x)\ge 0),x=a,x=b(a<b)$所围成：

    (1) 区域$D$绕$x$轴旋转一周所得到的旋转体积为：

    $V_D=2\pi\iint\limits_{D} r(x,y)d\sigma = 2\pi \iint\limits_{D} yd\sigma =2\pi \int_{a}^{b}dx\int_{0}^{f(x)} ydy=\pi \int_{a}^{b}f^2(x)dx$

    (2) 区域$D$绕$y$轴旋转一周所得到的旋转体积为：

    $V_D=2\pi\iint\limits_{D} r(x,y)d\sigma = 2\pi \iint\limits_{D} xd\sigma =2\pi \int_{a}^{b}dx\int_{0}^{f(x)} xdy=2\pi \int_{a}^{b}xf(x)dx$

（三）曲线弧长

  1. $C:y=y(x),a\le x\le b. \space s=\int_{a}^{b}\sqrt {1+y’^2}dx$

  2. $C:\left {\begin{aligned} x=x(t)\ y=y(t) \end{aligned} \right.,\alpha \le t\le \beta . \space s=\int_{\alpha}^{\beta}\sqrt {x’^2+y’^2}dt$

  3. $C:\rho=\rho(\theta),\alpha \le \theta \le \beta. \space s=\int_{\alpha}^{\beta}\sqrt {\rho ^2+\rho ‘^2}d\theta$

（四）旋转体侧面积

$S=2\pi \int_{a}^{b}f(x)\sqrt{1+f’^2(x)}dx$

反常积分敛散性

例题参考$660$上的第$483$题、第$596$题。

（一）无穷区间上的反常积分

  1. 定义法，直接求原函数带入

    $\int_{a}^{+\infty } f(x)dx=\lim\limits_{t \to +\infty} \int_{a}^{t}f(x)dx$或$\int_{-\infty}^{b} f(x)dx=\lim\limits_{t \to -\infty} \int_{t}^{b}f(x)dx$

    若上述表达式存在具体值则此反常积分收敛，反之发散。

    若$\int_{0}^{+\infty } f(x)dx$和$\int_{-\infty }^{0} f(x)dx$都收敛，则$\int_{-\infty}^{+\infty } f(x)dx$收敛。

  2. 结论法，熟记常用结论（无穷级数中的$p$级数）：

    $\LARGE \int_{a}^{+\infty } \frac{1}{x^p} dx;\left {\begin{aligned} P>1\space 收敛 \ P<1 \space 发散 \end{aligned} \right.$ $(a>0)$

（二）无界函数的反常积分

  1. 定义法，直接求原函数带入

    设$a$为$f(x)$的无界点，$\int_{a}^{b } f(x)dx =\lim\limits_{t \to a^+} \int_{t}^{b}f(x)dx$，若存在则收敛，反之发散。若无界点在区间中间，则分成两段计算考虑。

  2. 结论法，熟记常用结论（无穷级数中的$p$级数）：

    $\LARGE \int_{a}^{b } \frac{1}{(x-a)^p} dx;\left {\begin{aligned} P<1\space 收敛 \ P\ge 1 \space 发散 \end{aligned} \right.$

推广的积分中值定理：

设$f(x)$在$[a,b]$上连续，$g(x)$在$[a,b]$上可积且不变号，则至少存在一点$\xi \in (a,b)$，使得$\int_{a}^{b}f(x)g(x)dx=f(\xi)\int_{a}^{b}g(x)dx$。
